# More Integrals? (with powered trig functions?)

## Tips and Tricks

- Split odd powers of trig functions into products to find indents. 
- Use idents where possible
  - $sin^2x = (1-cos^2x)$
  - etc
    - get a trig idents table
- get half angle formulas
